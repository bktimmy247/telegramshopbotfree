# Roadmap bản trả phí / workshop

## 1. Thanh toán thật

- Tạo mã nạp `NAP...` duy nhất.
- Sinh VietQR theo số tiền + nội dung chuyển khoản.
- Nhận webhook SePay/Casso.
- Xác thực API key/signature.
- Chống xử lý trùng bằng `external_id`.
- Cộng tiền ví atomically.

## 2. Admin nâng cao

- Menu admin bằng button.
- Thêm/sửa/ẩn sản phẩm.
- Import stock/license hàng loạt.
- Xem đơn hàng.
- Broadcast khách hàng.
- Export báo cáo.

## 3. Chạy 24/7

- Webhook public qua Cloudflare Tunnel/domain.
- Deploy VPS.
- PM2/systemd.
- Backup database.
- Log/monitoring.

## 4. Cá nhân hóa ngành hàng

- Ebook/course/template.
- Bán key/license.
- Bán gói tư vấn.
- Bán API credit.
- Bán tài liệu nội bộ/khóa học cộng đồng.

## 5. Nâng cấp thành hệ sinh thái

- Landing page.
- Bot chăm sóc sau mua.
- Group/Zalo/Telegram cộng đồng.
- CRM mini.
- Affiliate/reseller API.
